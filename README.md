# Конфигурация Emacs для тех, кто не знает с чего начать

В этом репозитории хранятся настройки GNU Emacs для тех, кто хотел бы начать работу с редактором, но не знает, с чего начать.


## Установка

Emacs ищет свой конфигурационный файл в нескольких местах. Мы рекомендуем разместить файл `init.el` из этого репозитория в одном из каталогов:

* `~/.emacs.d/` — «старый» стиль.

* `~/.config/emacs/` — «новый» стиль, соответствующий спецификации XDG.
